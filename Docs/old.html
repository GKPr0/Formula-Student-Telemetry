Documentation for the CanReader app. The goal of this project is to create a desktop app, that will be collecting and visualizing real-time CAN data The app was created for Formula student team FS Tul Racing.

Objectives of the application¶

  1. View the current state of the vehicle
  2. View the continuous state of the vehicle
  3. Configurable CAN bus messages
  4. Connectivity status monitoring
  5. Adjustable saving of files
